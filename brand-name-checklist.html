Checklist

Brand name validation checklist

Before you publish a name, test it from several angles. This checklist is not legal advice, but it helps you catch common naming problems early.

Search and conflict checks

  • Search the exact name in major search engines and social platforms.
  • Check whether similar names already exist in the same category.
  • Look for unintended meanings, slang, or confusing associations.
  • Review domain availability and common social handle formats.

Audience and usability checks

  • Ask three people to spell the name after hearing it once.
  • Read it aloud in a sentence, such as “Find us at...” or “My channel is...”
  • Check whether the name still works if the project expands.
  • Compare it with two alternatives instead of judging it alone.

Decision rule

If a name is clear, distinct, flexible, and easy to repeat, keep it on the shortlist. If it needs a long explanation, it is usually better as an internal concept than a public name.
